# ProjetoSistemaAcademico


#Este projeto é um CRUD (Create, Read, Update, Delete) de um sistema acadêmico desenvolvido em Delphi, que gerencia informações de professores, estudantes e turmas. O sistema utiliza TStringList e TObjectList para armazenar dados em memória, com possibilidade de salvar e carregar arquivos de forma persistente.

-O projeto serve como base para estudos e prática de programação orientada a objetos em Delphi, manipulação de listas e interação com formulários.

Funcionalidades
-Gerenciamento de Estudantes

-Adicionar, editar e remover estudantes.

-Armazenamento de informações como nome, matrícula e idade.

-Exibição em um TStringGrid para visualização organizada.

-Gerenciamento de Professores

-Adicionar, editar e remover professores.

-Armazenamento de informações como nome, disciplina e matrícula.

-Lista organizada em TObjectList para manipulação de objetos.

-Gerenciamento de Turmas

-Criar turmas e associar estudantes e professores.

-Visualização das turmas e seus membros.

-Persistência de Dados

-Salvar dados em arquivos .txt utilizando TStringList.

-Carregar dados ao abrir o sistema, garantindo que as informações não se percam.

-Interface Gráfica

-Formulários claros e intuitivos.

-TStringGrid para exibição de listas.

-Botões para operações CRUD com confirmação de exclusão.
